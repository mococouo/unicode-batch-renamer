# GitHub Publishing Notes

## Repository Name

`unicode-batch-renamer`

## Short Description

No-dependency Python GUI/CLI for safely batch-renaming image and video files with Unicode templates and a multilingual interface.

## Suggested Topics

`python`, `tkinter`, `batch-rename`, `file-renamer`, `unicode`, `i18n`, `multilingual`, `images`, `videos`, `desktop-app`

## About

Unicode Batch Renamer is a small cross-platform Python tool for renaming image
and video files in bulk. It provides an English-first multilingual GUI, a simple
CLI, Unicode-safe naming templates, padded numbering, preview, and a two-step
rename process to reduce filename collisions.
